Gestión de Lista de Tareas (GUI con Tkinter)
Este proyecto es una aplicación de escritorio funcional desarrollada en Python utilizando la librería Tkinter. Permite gestionar tareas diarias a través de una interfaz gráfica intuitiva, diseñada como parte de una formación práctica en programación de interfaces.

Características
Añadir Tareas: Permite escribir una tarea y agregarla con un clic o presionando la tecla Enter.

Marcar como Completada: Las tareas terminadas cambian visualmente (color gris y prefijo ✔) mediante doble clic o el botón correspondiente.

Eliminar Tareas: Remueve tareas de la lista de forma sencilla.

Manejo de Eventos: Implementación de eventos de teclado y ratón para una experiencia fluida.

Requisitos
Lenguaje: Python 3.x

Librerías: tkinter (incluida en la instalación estándar de Python).

IDE Recomendado: PyCharm.

Instalación y Ejecución
Clonar el repositorio:

Bash
git clone https://github.com/kevinalex000-art/lista-de-tareas-gui.git
Ejecutar la aplicación:
Navega a la carpeta del proyecto y ejecuta el archivo principal:

Bash
python main.py
📂 Estructura del Proyecto
main.py: Contiene toda la lógica de la aplicación y la interfaz gráfica utilizando Programación Orientada a Objetos (POO).
